// 03c · DRIVERS

ghost-nvidia

RTX 5060 · BLACKWELL · DRIVERS

Instalador automático de drivers para la RTX 5060 (arquitectura Blackwell GB206). Esta GPU requiere el driver 570.169+ con módulos open — el driver propietario clásico NO es compatible. El script detecta y corrige automáticamente el problema de pantalla negra.

PROBLEMA CONOCIDO — PANTALLA NEGRA TRAS INSTALAR DRIVER
Causa: instalaste el driver propietario clásico (no el -open)
Causa: BIOS en modo Legacy/CSM en lugar de UEFI
Causa: kernel demasiado antiguo (necesitas 6.11+)
Causa: nouveau no estaba bloqueado antes de instalar nvidia

SOLUCIÓN AUTOMÁTICA
sudo bash ghost-nvidia.sh

El script hace en orden:
1. Verifica modo UEFI (avisa si estás en Legacy)
2. Verifica kernel 6.11+ (lo actualiza si es necesario)
3. Purga TODOS los drivers nvidia anteriores
4. Bloquea nouveau (conflicto directo con nvidia-open)
5. Instala nvidia-open-dkms + nvidia-driver 570.169+
6. Configura GRUB con nvidia-drm.modeset=1
7. Regenera initramfs
8. Reinicia (opcional)
VERIFICACIÓN POST-INSTALACIÓN
nvidia-smi → Debe mostrar RTX 5060 con driver 570+
lsmod | grep nvidia → nvidia_open debe aparecer
dmesg | grep -i nvidia → Sin errores NVRM

SI SIGUE EN PANTALLA NEGRA
Ctrl+Alt+F3 → acceder a TTY
sudo bash ghost-nvidia.sh --rescue → modo recuperación

// 05 · MANUAL

Documentación

GHOST EYES SYSTEM — MANUAL DE USUARIO v1.0

1. Instalación
Clonar el repositorio, ejecutar install.sh con privilegios de root. El script configura Plymouth, copia los dotfiles y registra los comandos ghost-* en el PATH del sistema.

2. Primera sesión
Al reiniciar verás la animación de arranque. La pantalla de login es negra con texto verde. Escribe tu usuario: notarás un sonido suave por tecla (via aplay + wav minimalista).

3. Uso de comandos
Escribe ghost-help en cualquier terminal para ver el listado completo con descripción. Cada comando tiene flag --verbose para más detalle y --json para exportar el resultado.

4. Personalización
Edita ~/.ghost-config para cambiar colores, deshabilitar sonidos o añadir tus propios alias.
