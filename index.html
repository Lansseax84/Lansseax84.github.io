// 02 · DESCRIPCIÓN

Programa

Este proyecto modifica un Kali Linux estándar para crear un entorno visual, accesible y temático. Incluye arranque personalizado con animación, pantalla de login táctica y comandos con salida formateada y coloreada para que cualquier persona pueda entender el output sin experiencia previa.

BASE

Kali Linux Rolling

Sistema operativo base sin modificaciones de kernel. Solo capa visual y scripts de usuario.

BOOT

Animación de arranque

GIF/Plymouth de La Parca al inicio. Login en negro con líneas verdes y sonido al escribir.

SHELL

Terminal personalizada

Prompt en naranja apagado, alias simplificados, outputs con tablas y colores claros.

// 05 · MANUAL

Documentación

GHOST EYES SYSTEM — MANUAL DE USUARIO v1.0

1. Instalación
Clonar el repositorio, ejecutar install.sh con privilegios de root. El script configura Plymouth, copia los dotfiles y registra los comandos ghost-* en el PATH del sistema.

2. Primera sesión
Al reiniciar verás la animación de arranque. La pantalla de login es negra con texto verde. Escribe tu usuario: notarás un sonido suave por tecla (via aplay + wav minimalista).

3. Uso de comandos
Escribe ghost-help en cualquier terminal para ver el listado completo con descripción. Cada comando tiene flag --verbose para más detalle y --json para exportar el resultado.

4. Personalización
Edita ~/.ghost-config para cambiar colores, deshabilitar sonidos o añadir tus propios alias.
